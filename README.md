**Program Details**

- The MotorPH Payroll System is a program designed for both employees and payroll staff. Users must enter the correct username and password to access the system. If the login credentials are incorrect, the system displays the message “Incorrect username and/or password.”
Employees can view their personal information by entering their employee number. The system displays their employee number, name, and birthday. If the employee number entered does not exist, the system shows the message “Employee number does not exist.”
Payroll staff have access to payroll processing for one employee or all employees. In the first cutoff, the system displays the employee number, name, birthday, total hours worked, gross salary, and net salary. In the second cutoff, the system shows the total hours worked, gross salary, deductions (SSS, PhilHealth, Pag-IBIG, and tax), total deductions, and net salary.
Overall, the program is developed and structured according to the requirements of MotorPH, ensuring that the payroll system processes and displays employee payroll information accurately and efficiently.
